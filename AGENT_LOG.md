# Agent Log

All agent sessions are recorded here. Each entry follows the format below.

## Entry Format

```
### [TIMESTAMP] — [AGENT-TYPE] — [MODULE-CODE]
- **Task ID:** TASK-NNN
- **Status:** COMPLETE | WIP | BLOCKED
- **Files changed:** list
- **Tests:** passed/total
- **Issues opened:** #N list (or NONE)
- **Issues closed:** #N list (or NONE)
- **Notes:** freeform
```
