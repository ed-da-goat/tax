## GEORGIA WITHHOLDING CALCULATION (Quick Formula)

```
Per-Period Withholding =
  ((Annual_Gross - Standard_Deduction - Personal_Exemptions) x Flat_Rate)
  / Pay_Periods_Per_Year

Where:
  Annual_Gross = Gross_Pay_Per_Period x Pay_Periods_Per_Year
  Standard_Deduction = from table above (by filing status)
  Personal_Exemptions = $2,700 (taxpayer) + $2,700 (spouse if MFJ)
                        + ($3,000 x dependents)
  Flat_Rate = 0.0519 (TY2025) or 0.0499 (TY2026 [UNVERIFIED])
  Pay_Periods_Per_Year = 52 (weekly), 26 (biweekly), 24 (semi-monthly),
                         12 (monthly)

If result is negative, withholding = $0.00
```
